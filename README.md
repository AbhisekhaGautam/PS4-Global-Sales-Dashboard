**📌 Project Overview**

This project showcases an end-to-end data analytics workflow using Power BI, starting from raw data cleaning to delivering actionable insights through an interactive dashboard.

The analysis focuses on global video game sales, helping identify trends, top-performing genres, and publisher dominance across different regions.


<br><br>

**🎯 Objectives**

Clean and preprocess raw gaming dataset
Analyze global and regional sales trends
Identify top genres and publishers
Build an interactive and visually appealing Power BI dashboard
Generate business insights from data


<br><br>

**📂 Dataset Details**

The dataset consists of the following columns:

Column Name	Description
Game	Name of the video game
Year	Release year
Genre	Game category
Publisher	Publishing company
North America	Sales in North America (millions)
Europe	Sales in Europe (millions)
Japan	Sales in Japan (millions)
Rest of World	Sales in other regions (millions)
Global	Total worldwide sales (millions)
🧹 Data Cleaning (Power Query)
<br><br>

**Performed in Power BI Power Query Editor:**
Removed NA values and blank rows
Fixed inconsistent data entries
Applied correct data types
Ensured data integrity across regional and global sales
Prepared dataset for efficient analysis
